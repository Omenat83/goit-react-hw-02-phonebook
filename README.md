# Task.
Напиши застосунок зберігання контактів телефонної книги.

Застосунок повинен складатися з форми і списку контактів. На поточному кроці реалізуй додавання імені контакту та відображення списку контактів. Застосунок не повинен зберігати контакти між різними сесіями (оновлення сторінки). Розшир функціонал застосунку, дозволивши користувачам додавати та видаляти номери телефонів. Додай поле пошуку, яке можна використовувати для фільтрації списку контактів за ім'ям. Заборони користувачеві можливість додавати контакти, імена яких вже присутні у телефонній книзі.
